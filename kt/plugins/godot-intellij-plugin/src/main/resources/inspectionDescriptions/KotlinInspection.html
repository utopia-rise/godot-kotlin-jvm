Runs Kotlin-specific Godot Kotlin/JVM inspections.

These checks operate on Kotlin PSI and K2 analysis to validate registration annotations, exported properties, RPC usage, signals, and callable references used in common Godot patterns.

Current checks include:

  • @RegisterProperty, @Export, and property hint validation
  • @RegisterFunction validation
  • @RegisterSignal validation
  • @Rpc annotation validation
  • callable-reference validation for signal, call, and rpc usage
