Runs JVM-facing Godot Kotlin/JVM inspections.

These checks focus on registered classes and other rules that can be validated from Java-shaped PSI, which makes them a good fit for Kotlin and potentially other JVM languages.

Current checks include:

  • @GodotScript usage validation
  • invalid Godot inheritance for registered classes
  • generic registered class rejection
  • default constructor requirements
  • duplicate registered class names
